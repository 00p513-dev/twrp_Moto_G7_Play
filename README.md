# TWRP Device configuration for Motorola Moto G7 Plus

## Device specifications

Basic   | Spec Sheet
-------:|:-------------------------
CPU     | Octa-core 2.2 GHz Cortex-A53
CHIPSET | Qualcomm SDM630 Snapdragon 630
GPU     | Adreno 508
Memory  | 4GB or 6GB
Shipped Android Version | 8.0 (Oreo)
Storage | 64GB or 128GB
Battery | 3200 mAh
Dimensions | 160 x 75.5 x 8 mm
Display | 1080 x 2160 pixels, 5.9" IPS LCD
Rear Camera  | 12 MP (f/1.7) + 5 MP (f/2.2), (PDAF, dual pixel)
Front Camera | 8 MP (f/2.2)

![Device Picture](https://cdn2.gsmarena.com/vv/pics/motorola/motorola-moto-g6-plus-2.jpg)

### Kernel Source
Check here: https://github.com/Jleeblanch/kernel_motorola_sdm660

### How to compile

```sh
. build/envsetup.sh
lunch omni_lake-eng
make -j4 recoveryimage
```
